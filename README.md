# 🧠 Fake Job Detection System

A machine learning-based web application that detects whether a job posting is fake or real.

## 🚀 Features
- NLP-based classification
- TF-IDF vectorization
- Flask web interface
- Real-time prediction

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- HTML/CSS

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
